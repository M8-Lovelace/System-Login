# System-login

Este paquete utiliza Vue 3, localstorage, pinia y Quasar Framework para crear una interfaz de usuario interactiva y atractiva.

## Instalación

Una pequeña introducción sobre la instalación.

1. Clone the repository: `git clone https://github.com/M8-Lovelace/System-Login.git`
2. Go to the folder: `cd System-Login`
3. Install dependencies: `npm install`
4. Run the project: `npm run dev`
